# Running the Portfolio Backend

Requirements:
- Node.js (v16+ recommended)

Install dependencies:

```bash
npm install
```

Start in development (auto-reload with nodemon):

```bash
npm run dev
```

Or start in production:

```bash
npm start
```

By default the server runs on port 3000. Open http://localhost:3000/portfolio.html to view the site.

Contact form submissions are stored in `data/messages.json` (this file is ignored by git via `.gitignore`).
